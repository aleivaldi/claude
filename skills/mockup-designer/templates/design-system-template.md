## Spacing

Spacing system based on 4px grid:

```

sm:  8px   // Compact spacing, form labels
md:  16px  // Standard spacing, component padding
lg:  24px  // Generous spacing, section gaps
xl:  32px  // Large spacing, major sections
2xl: 48px  // Extra large spacing, page sections
```

### Usage Guidelines

- Use consistent spacing throughout (stick to scale)
- Padding: typically md (16px) for components
- Margins: lg (24px) between major sections
- Forms: sm (8px) between label and input

## Components

### Button

**Variants**:

1. **Primary**: Main actions, CTAs
   - Background: Primary color
   - Text: White
   - Hover: Primary hover color + shadow-md

2. **Secondary**: Secondary actions
   - Background: Surface color
   - Text: Primary color
   - Border: 1px Border color
   - Hover: Background lightens, border → primary

3. **Ghost**: Tertiary actions
   - Background: Transparent
   - Text: Primary color
   - Hover: Background → Surface color

**Sizes**:
- Small: padding sm (8px) md (16px)
- Medium: padding sm (8px) md (16px) ← default
- Large: padding md (16px) lg (24px)

**States**:
- Default
- Hover (transform: translateY(-1px))
- Active (transform: translateY(0))
- Disabled (opacity: 0.5, cursor: not-allowed)
- Focus (outline: 2px solid primary, offset: 2px)

---

### Card

**Structure**:
- Background: Surface color
- Padding: lg (24px)
- Border radius: lg
- Shadow: sm (at rest), md (on hover if interactive)

**Variants**:
- Default: Static card
- Interactive: Hover effect (shadow-md, transform translateY)
- Outlined: Border instead of shadow

**Sub-components**:
- Card Header: border-bottom, padding-bottom md
- Card Title: H3 styling
- Card Content: Body text
- Card Footer: Optional actions area

---

### Input / Form Controls

**Input Field**:
- Padding: sm (8px) md (16px)
- Border: 1px Border color
- Border radius: md
- Font size: Body

**States**:
- Default
- Focus: Border → Primary, shadow ring (3px primary @ 10% opacity)
- Error: Border → Error color, shadow ring error
- Disabled: Background gray, cursor not-allowed

**Form Group**:
- Margin bottom: lg (24px)
- Label: Small size, font-weight 500, margin-bottom xs
- Helper text: Small size, Text secondary, margin-top xs
- Error text: Small size, Error color, margin-top xs

---

### Badge / Tag

**Purpose**: Status indicators, labels

**Variants**:
- Success: Light green background, dark green text
- Warning: Light yellow background, dark yellow text
- Error: Light red background, dark red text
- Neutral: Light gray background, dark gray text

**Style**:
- Padding: 4px 12px
- Border radius: sm or full (pill)
- Font size: 12px
- Font weight: 500

---

### Table

**Structure**:
- Container: Surface background, rounded corners, shadow-sm
- Header: Background color, font-weight 600, border-bottom 2px
- Row: Border-bottom 1px, hover background → Background color
- Cell padding: md (16px)

**Usage**:
- For data-heavy interfaces
- Include sorting indicators in headers
- Zebra striping optional (alternate row backgrounds)

---

### Modal / Dialog

**Structure**:
- Overlay: rgba(0,0,0,0.5) backdrop
- Container: Surface background, max-width 500px, shadow-lg
- Padding: xl (32px)
- Border radius: lg

**Components**:
- Header: H2 title, close button
- Content: Body text, forms, etc.
- Footer: Action buttons (right-aligned)
